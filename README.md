# EXERCICIOS-RACIOCINIO-ALGORITMICO
Este repositório reúne exercícios desenvolvidos ao longo do semestre na disciplina de Raciocínio Algorítmico.  
O objetivo é praticar lógica de programação, resolução de problemas e conceitos fundamentais da programação utilizando Python.
Os exercícios abordam temas como:
- Estruturas condicionais
- Estruturas de repetição
- Listas
- Matriz
- Dicionários
- Manipulação de dados
- Algoritmos básicos
- Entrada e saída de dados
Os exercícios estão separados por listas de atividades propostas durante a disciplina.
